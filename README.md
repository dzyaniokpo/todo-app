# Todo App

A beautiful, modern Todo list application built with React and Tailwind CSS.

## Features

- ✅ Add new tasks by typing and pressing Enter
- ✅ Mark tasks complete with a checkbox
- ✅ Delete tasks with an X button
- ✅ Dark mode toggle in the header
- ✅ Save everything to localStorage so it persists

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- LocalStorage API
